## Extension Name

Media Link Extractor

## Short Description (132 chars max)

Extracts video download links from web pages. Select files, choose a folder, and download them all with one click.

## Full Description

Media Link Extractor automatically scans web pages for video download links and presents them in a clean, organized list. Select the files you want, specify a download subfolder, and download them all sequentially — no more clicking links one by one.

**Key Features:**

• Automatically detects video links on any web page
• Supports .mp4, .mkv, .avi, .webm, .mov, .flv, .wmv, .m4v, .ts, and .3gp formats
• Smart detection — finds video URLs in HTML elements, data attributes, query parameters, and inline scripts
• Checkbox selection with Select All / Deselect All
• Download to a custom subfolder inside your Downloads directory
• Remembers your folder preference between sessions
• Sequential downloading — queues files and downloads them one at a time
• Live progress tracking with per-file status indicators
• Minimal permissions — only accesses a page when you click the extension icon
• No data collection — completely private, everything stays on your device

**How to Use:**

1. Visit any web page with video download links
2. Click the Media Link Extractor icon in the toolbar
3. Select the files you want to download
4. Optionally enter a subfolder name
5. Click "Download Selected"

**Tip:** For the best experience, go to edge://settings/downloads and turn off "Ask me what to do with each download" so files download automatically without prompts.
